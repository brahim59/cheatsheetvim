Commande VIM

Vim est un éditeur de texte, c’est-à-dire un logiciel permettant la manipulation de fichiers texte. Il est directement inspiré de vi (un éditeur très répandu sur les systèmes d’exploitation de type UNIX), dont il est le clone le plus populaire. Son nom signifie d’ailleurs Vi IMproved, que l’on peut traduire par « VI aMélioré ».

Presentation

Dans un terminal informatique, Vim fonctionne en mode texte plein écran, comme l’éditeur de texte vi. Il propose énormément de fonctions qui sont, pour la grande majorité, accessibles au clavier.

Differents Modes

Vim est un éditeur modal. Cela signifie que l’on effectue différentes tâches dans différents modes

  1. Mode normal

    Vim démarre en mode Normal, aussi appelé mode Commande. Dans ce mode, il est par exemple possible de copier des lignes ou de les déplacer grâce à des raccourcis, de mettre du texte en forme, ou de se déplacer dans le fichier.

  2. mode insertion

    Le mode Insertion est le plus naturel, car il se comporte comme la plupart des autres éditeurs. On peut passer du mode Normal au mode Insertion avec la commande i (comme « Insertion »)

  3. Mode Ligne-de-commande

    On atteint ce mode en tapant d’abord le caractère deux-points « : ». Ensuite, un certain nombre de commandes peuvent être saisies.

Principale commandes

Commandes principales
Commande Sens Type Effet
i insert passe en mode insertion
a append passe en mode insertion en faisant avancer le curseur
o open passe en mode insertion en créant une nouvelle ligne sous le curseur
R open passe en mode remplacement
A passe en mode insertion en allant à la fin de la ligne
v visual passe en mode visuel
